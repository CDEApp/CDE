Welcome to cdeWeb

This is a web interface for searching catalog files created by cde.

"cde" is an abreviation for "catalog directory entries".


This application contains goodness from the following development resources.

  • AngularJS
  • AngularUI
  • Twitter Bootstrap
  • Font Awesome
  • ASP.NET MVC
  • autofac
  • SignalR
